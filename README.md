# Doctor Appointment Website ü•º
A doctor appointment website build with mongodb, express, reactjs and nodejs.

# üí°About this project 
Doctor Appointment website helps to appoint Doctors. This project consists of 3 parts i.e. Backend, Frontend and Admin. User can easily login and register. There are 4 sections i.e. Home page, Doctors, About and Contact us. Upon clicking on doctors section you will see various doctors with there specialization. After chossing doctor you will go to doctor personal page and there you can book appointment. In case if you want to cancel your appointment, you can. Admin can add new doctors.
